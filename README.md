# DMBS_Project
## Decision Making Bash Script

`g++ -I/usr/include/cppconn -o fLUp fareList_Updater.cpp -L/usr/lib -lmysqlcppconn -ljsoncpp`

`./fLUp`

`g++ -I/usr/include/cppconn -o run main.cpp -L/usr/lib -lmysqlcppconn -ljsoncpp`

### To Run 

`./run`
